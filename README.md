# caddy.posixism
Reimplementation of https://github.com/shinh/caddy in POSIXism shell script

Has several shell scripts.

# Requirements
- Utilities defined in POSIX standards directly.
- bc, for mime-make.
- One of wget or curl, for POSTing.
